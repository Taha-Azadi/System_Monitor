# Usage Guide

## Getting Started

Launch the application by running:
```bash
python main.py
```

## Navigation

Use the sidebar to switch between different monitoring views:

- **Dashboard** - System overview with live charts
- **Processes** - Manage running processes
- **Performance** - Detailed CPU, Memory, Disk metrics
- **Network** - Network interfaces and connections
- **Storage** - Disk usage and I/O statistics
- **GPU** - Graphics card monitoring
- **Services** - Windows services management
- **Startup** - Startup programs control
- **System Info** - Hardware information
- **Settings** - Application configuration

## Process Management

1. Select a process from the list
2. Click "End Task" to terminate it
3. Use the search box to filter processes
4. Click column headers to sort

## Settings

Configure themes, alert thresholds, and update intervals in the Settings panel.
