# System Monitor Pro

A powerful, modern system monitoring tool built with Python and CustomTkinter. Better than Task Manager.

## Features

- **Real-time Dashboard** - Beautiful overview with live metric cards and performance charts
- **Process Manager** - Full process control: view, search, sort, kill, and change priorities
- **Performance Monitor** - Detailed CPU per-core usage, memory breakdown, disk I/O
- **Network Monitor** - Real-time network speed, interface details, active connections
- **Storage Monitor** - Disk partitions, usage statistics, I/O counters
- **GPU Monitor** - NVIDIA/AMD GPU usage, memory, temperature monitoring
- **Services Manager** - Windows services control (start, stop, restart)
- **Startup Manager** - Manage startup programs and registry entries
- **System Information** - Complete hardware and OS details
- **Customizable Settings** - Themes, alerts, thresholds
- **Modern UI** - Dark/Light themes with CustomTkinter
